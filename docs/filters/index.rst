
Filters
=======

DataSifter is a module for filtering text and numbers.
With filtering, you can simplify your conditionals so you don't have to write huge checks

As of version 0.0.5.1 (latest), the following filters are available:
.. toctree::

    "is_url()" filter
    "contains()" filter
    "regexp_matches()" filter
    "in_range()" filter
    "length()" filter
    "length_is()" filter
    "file_is()" filter
    "is_bad_words()" filter
    "is_request()" filter
